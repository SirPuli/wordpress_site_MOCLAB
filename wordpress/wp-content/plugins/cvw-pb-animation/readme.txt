Add animation to widgets and rows in Siteorigin Page Builder.

== Description ==

This plugin adds customizable animation to any widget and row of SiteOrigin Page Builder.

It's based on free AOS animation library. You can set:
- Animation type.
- Duration.
- Delay.
- Easing.
- Anchor.


== Installation ==

1. Upload cvw-pb-animation folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. It doesn't have any page configuration.

== Configuration ==

Once the extension is installed go to a page or post built with Page Builder and will find the animation options.

Edit the widget or row, and in the right sidebar you will see a new menu element "AOS Animation". There you can set:
- Animation type.
- Duration.
- Delay.
- Easing.
- Anchor.
- Repeat.
